## Installation

VEUI is available through [VDPod](http://gitlab.ivedeng.com/veui/vdpod). To install it, simply add the following line to your Podfile:

```ruby
source 'http://gitlab.ivedeng.com/veui/vdpod.git'

pod 'VEUI'
```
